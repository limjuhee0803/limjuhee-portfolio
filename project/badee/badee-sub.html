challenge

project goal

Badee 브랜드가 가진 유니크한 분위기를 더욱 살릴 수 있도록 리디자인 하였습니다.
브랜드의 이미지가 화려하여 레이아웃은 심플하게, 하지만 동적인 움직임을 추가하여 재미요소를 주었습니다.

직선의 블랙 라인과 흐르는 텍스트들을 적극적으로 활영하여 너무 단조롭지 않도록 하였으며,
많은 이미지를 효과적으로 보여주기 위하여 부분 스크롤을 적극적으로 활용하였습니다.

  • 유니크한 디자인
  • 깔끔한 레이아웃과 동적인 움직임
  • 기존 사이트의 ui/ux 개선

design

design element

color

  • # 000000
  • # 262626
  • # F8BB0A
  • # FFFFFF

기본적인 컬러는 블랙과 화이트만을 사용하지만, hover시에는 채도가 강한 형광 컬러를 포인트로 주어 화려한 느낌 강조

font

국문은 NotoSanskKR, 영문은 볼드한 느낌이 강한 Russo One 폰트를 사용

challenge

problem & solution

publishing

  • 부분 스크롤 활용
  • 마우스 좌표를 이용한 이벤트
  • addClass를 활용한 다양한 효과

problem & solution

  • 1. 부분 스크롤의 영역이 좁아 스크롤이 어려움
    • - padding값을 활용하여 부분스크롤이 가능한 범위는 넓힘
    • - 바뀐 레이아웃은 transform으로 조정
  • 2. 하단의 내 포트폴리오 바로가기가 연결되지 않음
    • - .attr('href','링크')를 사용하였지만 링크가 생성되지 않음
    • - 태그 자체가 a태그여야 식이 성립되 된다는 것을 알게됨
    • - 태그를 수정하자, 곧바로 정상적으로 링크가 연결됨

challenge

project goal

Jeep 브랜드가 가진 무거운 분위기를 더욱 살릴 수 있도록 리디자인 하였습니다.
또한 복잡한 이미지를 다루는 만큼 최대한 깔끔한 레이아웃으로 구성하여 시각적인 편안함을 주었습니다.

검정과 명도대비가 강한 노란색을 포인트 컬러를 중간중간 활용하여 너무 단조롭지 않도록 하였으며,
곳곳에 마우스 이벤트를 활용하여 디자인에 활기를 주었습니다.

  • 무게감 있는 디자인
  • 깔끔한 레이아웃
  • 기존 사이트의 ui/ux 개선

design

design element

color

  • # 000000
  • # 262626
  • # F8BB0A
  • # FFFFFF

무거운 분위기가 매력적인 Jeep 브랜드의 아이덴티티를 살려 블랙과 다크그레이를 메인으로 디자인한 뒤, 옐로우 컬러를 포인트 컬러로 사용

font

국문은 NotoSanskKR, 영문은 Roboto Flex를 사용하고, 배너나 베스트 모델 등 강조할 때는 Druck Trial을 사용

challenge

problem & solution

publishing

  • root 변수 활용
  • 가변 단위 활용
  • grid와 flex를 이용한 레이아웃 구축

problem & solution

  • 1. header영역의 높이가 높아 페이지를 너무 가림
    • - header영역의 높이를 약간 수정하였지만, 더욱 효과적인 방법이 필요
    • - if문을 활용하여 스크롤을 내리면 header영역이 올라가고, 스크롤을 올리면 header영역이 내려오게 수정
  • 2. html을 validator 검사 시 이유를 알 수 없는 오류가 뜸
    • - 영역을 나누기 위하여 div와 section 태그를 많이 활용
    • - div안에는 section이 가능하지만 section안에는 div사용이 불가능하다는 것을 알게됨
    • - 태그를 수정하자 오류가 정상적으로 사라짐

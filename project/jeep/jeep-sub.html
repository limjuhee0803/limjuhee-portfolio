challenge

project goal

Jeep 브랜드가 가진 무거운 분위기를 더욱 살릴 수 있도록 리디자인 하였습니다.
또한 복잡한 이미지를 다루는 만큼 최대한 깔끔한 레이아웃으로 구성하여 시각적인 편안함을 주었습니다.

검정과 명도대비가 강한 노란색을 포인트 컬러를 중간중간 활용하여 너무 단조롭지 않도록 하였으며,
곳곳에 마우스 이벤트를 활용하여 디자인에 활기를 주었습니다.

  • 무게감 있는 디자인
  • 깔끔한 레이아웃
  • 기존 사이트의 ui/ux 개선

design

design element

color

  • # 000000
  • # 262626
  • # F8BB0A
  • # FFFFFF

무거운 분위기가 매력적인 Jeep 브랜드의 아이덴티티를 살려 블랙과 다크그레이를 메인으로 디자인한 뒤, 옐로우 컬러를 포인트 컬러로 사용

font

국문은 NotoSanskKR, 영문은 Roboto Flex를 사용하고, 배너나 베스트 모델 등 강조할 때는 Druck Trial을 사용

challenge

problem & solution

publishing

  • @keyframe을 이용한 애니메이션
  • Mediaquery를 활용한 반응형 웹
  • grid와 flex를 이용한 레이아웃 구축

problem & solution

  • 1. 메인 배너의 도넛 이미지가 디스플레이의 사이즈에 따라 위치 변화
    • - 기존의 % 값이었던 메인 배너의 높이 값을 고정 값을 px로 수정하여 높이 값 고정
    • - 기타 margin이나 padding 값 또한 고정값으로 수정한 뒤, overflow:hidden; 적용
  • 2.공식 이미지 자료 부족으로 인한 디자인의 어려움
    • - 공식 한국 사이트 외에도 해외의 공식 사이트들을 찾아 이미지를 적용
    • - 유사한 스타일의 도넛 이미지를 수정하여 이미지 적용
